# Mobile App A/B Testing & Retention Analysis

## О проекте
Это мой самостоятельный пет-проект для практики работы со статистикой и продуктовыми метриками. 
В нем я анализирую результаты A/B теста нового экрана онбординга в мобильном приложении, чтобы понять, влияет ли он на первичную конверсию и возвращаемость пользователей на 7-й день (Retention Day 7).

## Инструменты
* **Python:** Pandas, NumPy для манипуляции данными.
* **Статистика:** `scipy.stats`, `statsmodels` (Z-test для пропорций).
* **Визуализация:** Matplotlib, Seaborn.

## Методология A/B Теста
Для проверки гипотезы о том, что новый онбординг увеличивает конверсию, использовался Z-критерий для разности двух долей. Формула тестовой статистики:

$$Z = \frac{\hat{p}_1 - \hat{p}_2}{\sqrt{\hat{p}(1-\hat{p})(\frac{1}{n_1} + \frac{1}{n_2})}}$$

Где $\hat{p}_1$ и $\hat{p}_2$ — выборочные конверсии в группах, а $\hat{p}$ — общая конверсия. Уровень значимости ($\alpha$) установлен на 5%.

## Результаты и выводы
1. **Conversion Rate (CR):** В тестовой группе конверсия составила ~15% против ~12% в контрольной. 
2. **Статистическая значимость:** P-value составил менее 0.05. Разница статистически значима, внедрение нового экрана оправдано.
3. **Влияние на Retention:** Пользователи, прошедшие новый онбординг, показали корреляционный рост Retention 7-го дня, так как более осознанно подошли к использованию продукта.
